

#  Library Management System (VB.NET)

##  Overview

The **Library Management System** is a desktop application developed using **Visual Basic .NET (VB.NET)**.
It is designed to help manage library operations efficiently, including book records, member management, and book lending/return processes.

This project was developed for **academic purposes** and demonstrates practical use of **VB.NET**, **Windows Forms**, and basic database-driven application concepts.
