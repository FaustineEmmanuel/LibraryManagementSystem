
#  Library Management System (VB.NET)

##  Overview

The **Library Management System** is a desktop application developed using **Visual Basic .NET (VB.NET)**.
It is designed to help manage library operations efficiently, including book records, member management, and book lending/return processes.

This project was developed for **academic purposes** and demonstrates practical use of **VB.NET**, **Windows Forms**, and basic database-driven application concepts.



## Features

*  Manage library books (add, update, delete)
*  Manage library members
*  Issue and return books
*  Track borrowed and returned books
*  User-friendly Windows Forms interface
*  Structured and maintainable project design



##  Technologies Used

* **Programming Language:** Visual Basic .NET (VB.NET)
* **Framework:** .NET Framework
* **IDE:** Visual Studio 2022
* **Application Type:** Windows Forms Application

##  How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/FaustineEmmanuel/VB-LibraryManagementSystem-2026.git
   ```
2. Open **Visual Studio 2022**
3. Click **File → Open → Project/Solution**
4. Select `LibraryManagementSystem.sln`
5. Click **Start (▶)** to run the application
